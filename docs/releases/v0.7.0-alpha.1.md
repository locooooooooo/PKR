# PKR v0.7.0-alpha.1

PKR v0.7.0-alpha.1 is a local public alpha of the PKR framework and reference
Runtime.

## Boundary

- `pkr init`, `pkr run`, and `pkr status` are the supported first-use path.
- Provider callbacks are non-authoritative work reports.
- A distinct Repository Verifier collects Git and process evidence.
- Runtime recomputes evidence and creates acceptance only after a passing
  verification.
- Failed verification evidence remains durable and blocks the Task.

## Prerequisites

- Node.js 24 or newer;
- Git;
- Python 3.11 or newer for conformance checks;
- an installed and authenticated Codex CLI for `pkr run`.

Provider and verification commands are trusted host processes. This alpha does
not provide an OS sandbox, cloud Provider adapters, automatic model selection,
or production stability guarantees. v0.8 and v0.9 evolution work is not
included.

The npm package is not published in this candidate. Use the source install in
the README until an authenticated npm publication has passed its install and
CLI smoke checks.
